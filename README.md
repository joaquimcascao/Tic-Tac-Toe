# Tic Tac Toe — Console

A simple Tic Tac Toe game that runs in the terminal using plain JavaScript and Node.js.

## Overview
This project implements the classic Tic Tac Toe game logic for two players in a terminal environment.
It focuses on fundamental programming concepts such as state management, control flow, input validation, and win/draw detection.

## Features
Two-player turn-based gameplay (X and O)
Win detection (rows, columns, diagonals)
Draw detection
Game state management

This project was created to practice fundamental programming concepts and version control using Git and GitHub.
